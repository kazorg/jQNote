jQuery Note

Example 1

$.note("Just a string with default settings! I'll go away after 5 seconds or when clicked.");

Example 2

$.note({
	message: "I'm a message and will only leave if clicked.",
	autoHide: false,
	clickToClose: true
});

Example 3

$.note({
	message: "I'm a message and will stay here for 8 seconds then remove myself. You can't click me to dismiss. Also, slow transition.",
	autoHide: true,
	hideDelay: 8000,
	clickToClose: false,
	transitionSpeed: 900
});

Defaults

{
	transitionSpeed: 800, // Speed of transition (in/out).
	autoHide: false, // Hides automatically after hideDelay amount has passed.
	hideDelay: 5000, // autoHide delay. Not used if "autoHide: false".
	clickToClose: true, // Click to close dialog
	message: 'No message specified' // Message to show in dialog
}

Change defaults with $.noteSetup()

e.g. change hideDelay to 1000ms:

$.noteSetup({ hideDelay: 1000 });
(then go press the "Example 1" button).

Anything specified with $.noteSetup() will overwrite corresponding default value if it's not specified separately in $.note() (e.g. $.note({hideDelay: 300}) will overwrite $.noteSetups hideDelay value).
